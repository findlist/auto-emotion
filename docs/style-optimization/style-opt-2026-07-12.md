## 具体修改内容

### 1. 首页：雷达卡片标题层次增强

**文件**: `src/pages/home.tsx`

**问题**: "压力分布雷达"标题使用 `font-cn text-lg` 单行展示，缺乏副标说明，与下方圆点 + 条纹装饰背景的视觉重量不匹配。

**修改**: 标题升级为标题 + 副标两行结构，字号提升至 `text-xl`

**优化前**:
```tsx
<p className="font-cn text-lg mb-3 text-center">压力分布雷达</p>
```

**优化后**:
```tsx
<div className="text-center mb-3">
  <p className="font-cn text-xl">压力分布雷达</p>
  <p className="font-mono text-xs text-cream/60 mt-0.5">实时监测多维情绪压力</p>
</div>
```

**效果**: 标题层次更完整，副标以 `text-cream/60` 浅色辅助说明雷达图用途，与装饰背景共同营造氛围。

### 2. 大厅：角色信息卡 grid 布局重构

**文件**: `src/pages/lobby.tsx`

**问题**: 角色信息卡 4 行垂直堆叠（昵称/等级/金币/战力），每行结构相同视觉单调，信息密度低，卡片高度偏大。

**修改**: 昵称保留独占一行，等级/金币/战力三项核心数据 grid 3 列排列，每项以浅色底块包裹并使用对应主题色加粗数值。

**优化前**:
```tsx
<div className="text-left space-y-2 font-mono text-sm pl-2">
  <p className="text-ink"><span className="text-ink/60">昵称:</span> {user.nickname}</p>
  <p className="text-ink"><span className="text-ink/60">等级:</span> Lv.{user.level}</p>
  <p className="text-ink"><span className="text-ink/60">金币:</span> <span className="text-yellow font-bold">{user.coins}</span></p>
  <p className="text-ink"><span className="text-ink/60">战力:</span> <span className="text-mint font-bold">{user.power}</span></p>
</div>
```

**优化后**:
```tsx
<div className="text-left font-mono text-sm pl-2">
  <p className="text-ink mb-3">
    <span className="text-ink/60">昵称:</span> <span className="font-bold">{user.nickname}</span>
  </p>
  <div className="grid grid-cols-3 gap-2">
    <div className="bg-ink/5 px-2 py-1.5 rounded">
      <p className="text-ink/60 text-xs">等级</p>
      <p className="text-ink font-bold">Lv.{user.level}</p>
    </div>
    <div className="bg-yellow/20 px-2 py-1.5 rounded">
      <p className="text-ink/60 text-xs">金币</p>
      <p className="text-yellow font-bold">{user.coins}</p>
    </div>
    <div className="bg-mint/20 px-2 py-1.5 rounded">
      <p className="text-ink/60 text-xs">战力</p>
      <p className="text-mint font-bold">{user.power}</p>
    </div>
  </div>
</div>
```

**效果**:
- 三项数据用浅色底块（`bg-ink/5`、`bg-yellow/20`、`bg-mint/20`）区分，主题色与数值色呼应
- 卡片高度降低，信息密度提升
- 数据块视觉层次：label 用 `text-xs` 弱化、数值用 `font-bold` 强化

### 3. 房间：玩家行 avatar + 状态本地化

**文件**: `src/pages/room.tsx`

**问题 A**: 玩家行仅显示昵称文字，缺乏视觉识别度。首页、挂机页已有圆形首字母 avatar，房间页玩家行未对齐。

**问题 B**: 状态字段直接显示 `roomStore.status`（英文 `waiting`/`playing` 等），用户对内部协议值无感知。

**修改 A**: 每个玩家行左侧添加 8x8 圆形首字母 avatar，按角色差异化配色：
- 房主 → `bg-yellow text-ink`
- 当前用户 → `bg-pink text-cream`
- 其他玩家 → `bg-ink text-cream`

**修改 B**: 新增 `ROOM_STATUS_LABEL` 映射常量，将 6 种状态英文值映射为中文：

```tsx
const ROOM_STATUS_LABEL: Record<string, string> = {
  waiting: '等待中',
  ready: '已准备',
  generating: '生成中',
  playing: '游戏中',
  settling: '结算中',
  closed: '已关闭',
};
```

显示时：`{ROOM_STATUS_LABEL[roomStore.status] ?? roomStore.status}`（fallback 保留原值，未知状态不丢失）

**效果**:
- 玩家行视觉与首页/挂机页头像风格统一，房主/自己/他人三色 avatar 一眼可辨
- 状态字段从英文协议值变为用户可读的中文标签

### 4. 战斗：结算弹窗前 3 名差异化背景

**文件**: `src/pages/battle.tsx`

**问题**: 结算弹窗排名列表仅 `idx === 0` 有 `border-yellow bg-yellow/10` 高亮，第 2/3 名与普通排名视觉一致，未利用金/银/铜三色体系。原 `medalClass` 函数仅用文字色区分（`text-yellow`/`text-gray-400`/`text-amber-700`），背景一致导致层次不强。

**修改**: 扩展排名行的背景与边框差异化，与排行榜 `getRowStyle` 函数对齐：

**优化前**:
```tsx
className={`flex justify-between items-center px-3 py-1.5 border-2 ${
  idx === 0 ? 'border-yellow bg-yellow/10' : 'border-ink/20'
}`}
```

**优化后**:
```tsx
className={`flex justify-between items-center px-3 py-1.5 border-2 ${
  idx === 0
    ? 'border-yellow bg-yellow/10'
    : idx === 1
    ? 'border-gray-400 bg-gray-100'
    : idx === 2
    ? 'border-amber-700 bg-amber-50'
    : 'border-ink/20'
}`}
```

**效果**: 前 3 名的边框色 + 背景色形成金/银/铜三档视觉差异，与排行榜页 `getRowStyle` 保持一致，跨页面奖牌语义统一。

### 5. 挂机：战力数值框 padding 加大

**文件**: `src/pages/idle.tsx`

**问题**: 战力展示区属性框 `px-2 py-1` padding 较紧凑，label 与数值贴近边框，视觉呼吸感不足。

**修改**: padding 由 `px-2 py-1` 提升为 `px-2.5 py-1.5`

**优化前**:
```tsx
<div className="bg-cream/10 px-2 py-1 rounded border border-cream/20">
```

**优化后**:
```tsx
<div className="bg-cream/10 px-2.5 py-1.5 rounded border border-cream/20">
```

**效果**: 4 个属性框（攻击/防御/生命/效率）padding 增加 25%，label 与数值距边框留白更舒适。

### 6. 商店：类型筛选非激活态对比度增强

**文件**: `src/pages/shop.tsx`

**问题**: 商品类型筛选 tab 非激活态使用 `bg-ink/10 text-ink/70`，10% 透明度底色与 cream 背景对比度低，非激活 tab 几乎"消失"。

**修改**: 提升底色与文字透明度

**优化前**:
```tsx
'bg-ink/10 text-ink/70 hover:bg-ink/20'
```

**优化后**:
```tsx
'bg-ink/15 text-ink/80 hover:bg-ink/25'
```

**效果**: 非激活 tab 底色加深 5%、文字加深 10%，与 cream 背景形成更清晰的可辨度，hover 区间同步加深保持递进。

### 7. 任务：进度条高度增加

**文件**: `src/pages/tasks.tsx`

**问题**: 任务进度条 `h-2`（8px）较细，`progress-fill` 流光效果在窄条上不明显，进度可视化弱。

**修改**: 高度 `h-2` → `h-2.5`（10px）

**优化前**:
```tsx
<div className="h-2 bg-ink/20 rounded-full mb-3" role="progressbar" ...>
```

**优化后**:
```tsx
<div className="h-2.5 bg-ink/20 rounded-full mb-3" role="progressbar" ...>
```

**效果**: 进度条加高 25%，流光高光条更显眼，任务进度反馈更直观。成就页进度条未修改（保持 `h-2` 以与卡片紧凑度匹配）。

### 8. 赛季通行证：等级徽章尺寸增大 + 解锁环

**文件**: `src/pages/season-pass.tsx`

**问题**: 等级圆形徽章 `w-12 h-12`（48px）尺寸偏小，`font-mono font-bold` 无字号指定，与右侧"等级 X"标题文字重量不匹配。解锁态仅有 mint 底色 + 阴影，缺乏"已解锁"的视觉强调。

**修改**: 尺寸 `w-12 h-12` → `w-14 h-14`（56px），新增 `text-lg` 字号，解锁态叠加 `ring-2 ring-yellow/40` 黄色光环

**优化前**:
```tsx
<div className={`w-12 h-12 rounded-full flex items-center justify-center font-mono font-bold ${
  isUnlocked ? 'bg-mint text-ink shadow-[2px_2px_0_#1a1a1a]' : 'bg-ink/20 text-ink/50'
}`}>
```

**优化后**:
```tsx
<div className={`w-14 h-14 rounded-full flex items-center justify-center font-mono font-bold text-lg ${
  isUnlocked
    ? 'bg-mint text-ink shadow-[2px_2px_0_#1a1a1a] ring-2 ring-yellow/40'
    : 'bg-ink/20 text-ink/50'
}`}>
```

**效果**:
- 徽章尺寸增加 17%，数字字号升至 `text-lg`，与"等级 X"标题字号更协调
- 解锁态叠加黄色光环（`ring-yellow/40`），与赛季通行证"高级感"调性呼应
- 未解锁态保持灰度，解锁/未解锁对比更鲜明
