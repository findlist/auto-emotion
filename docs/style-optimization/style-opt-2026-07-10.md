## 二、全局 CSS 扩展（index.css）

### 新增关键帧动画

| 动画名 | 用途 |
|--------|------|
| `staggerIn` | 列表/卡片从下淡入交错入场 |
| `slideInLeft` | 水平列表从左淡入 |
| `popIn` | 弹窗放大淡入（带回弹） |
| `progressShimmer` | 进度条流光高光滑动 |
| `badgePulse` | 徽章脉冲光环（Top1/重要提示） |
| `spinSlow` | 装饰元素缓慢旋转 |

### 新增工具类（@layer utilities）

| 类名 | 用途 |
|--------|------|
| `.animate-stagger` | 交错入场动画（配合 delay-* 类） |
| `.animate-pop` | 弹窗放大入场 |
| `.animate-slide-left` | 从左淡入 |
| `.animate-spin-slow` | 缓慢旋转 |
| `.animate-badge-pulse` | 徽章脉冲 |
| `.delay-50` ~ `.delay-700` | 交错延迟工具类 |
| `.bg-stripes` | 对角条纹装饰背景 |
| `.bg-dots` | 圆点网格装饰背景 |
| `.bg-glow-pink` | 左上角光晕氛围背景 |
| `.scrollbar-brutal` | Neo-brutalism 风格自定义滚动条 |

### 新增组件类（@layer components）

| 类名 | 用途 |
|--------|------|
| `.card-hover` | 卡片悬停浮起效果（负向位移 + 阴影扩大） |
| `.progress-fill` | 进度条流光覆盖层（::after 高光条） |
| `.medal-gold` | 金牌渐变 + 金属质感阴影 |
| `.medal-silver` | 银牌渐变 + 金属质感阴影 |
| `.medal-bronze` | 铜牌渐变 + 金属质感阴影 |
| `.hud-chip` | 战斗 HUD 毛玻璃信息条（backdrop-blur + 描边） |

## 四、优化前后对比总结

### 视觉层次提升
- **优化前**：大量页面使用相同的 `shadow-[3px_3px_0_#1a1a1a]` + `border-2 border-ink`，缺乏层次区分
- **优化后**：通过 `bg-glow-pink` 头部氛围、`bg-stripes`/`bg-dots` 装饰纹理、`drop-shadow` 标题阴影、差异化边框色等手段建立多层视觉层次

### 交互体验提升
- **优化前**：卡片无 hover 反馈，按钮仅 hover 变色
- **优化后**：`card-hover` 统一卡片浮起效果，按钮加 `active:translate-x/y` + `active:shadow-none` 按下位移，输入框加 `focus:ring`

### 入场动画
- **优化前**：页面内容直接显示，无过渡
- **优化后**：`animate-stagger` + `delay-*` 交错入场，弹窗/遮罩用 `animate-pop` 带回弹放大

### 游戏沉浸感
- **优化前**：战斗 HUD 为纯色矩形，排行榜 Top3 仅底色区分
- **优化后**：`hud-chip` 毛玻璃 HUD、`medal-gold/silver/bronze` 金属质感奖牌、`animate-badge-pulse` 冠军脉冲光环、`progress-fill` 进度条流光

### 状态反馈
- **优化前**：加载/空状态为纯文字"加载中..."/"暂无数据"
- **优化后**：自定义旋转环 spinner + 文字提示，空状态加 `animate-bounce-slow` + 补充说明文字

### 一致性
- **优化前**：各页面按钮、卡片、标签样式不统一
- **优化后**：通过 `card-hover`、`progress-fill`、`scrollbar-brutal`、`bg-glow-pink` 等可复用组件类统一风格，所有按钮统一按下效果，所有 Tab 统一 `border-b-3` 激活态
